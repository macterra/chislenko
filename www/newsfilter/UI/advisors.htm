Edit/Select Favorite Advisors Dialogue.

This is the settings form which allows users to edit/choose their favorite advisors.

In the left form part the list of advisors for the active interest is represented.   (An interest is considered to be active when chosen in the Settings form)   The user can select some/all of them by pressing buttons ">>" & "<<".  Button "People respected by advisors" returns the list of advisors prefered to the advisors chosen in the list.  Button "Find like-minded people" returns the list of advisors with settings closest to the user's settings.

If check box "Use automatic selection" is active, the user instead of   selecting advisors from the list, gets a standart list of either the most rated advisors or the most popular advisors, depending on which one of the two radio-buttons is on.

Button "Ok" returns the user back to the "Settings" form, having admitted the setting change;  button "Cancel" does the same, but without a advisor setting change.
