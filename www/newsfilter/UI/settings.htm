Edit/Select Settings Dialogue.

This is the settings form which allows users to edit/choose their favorite settings.

In the top-left corner there are three buttons which allow the user to select the type of the interest tree performance--Panel, Tree, or Altavista like.  The three samples of these tree performances are available at the following destinations:  www.e-spaces.com/unnf/ui1.jpg, www.e-spaces.com/unnf/ui2.gif, www.e-spaces.com/unnf/ui3.gif  In the current form the Tree performance is shown.  By pressing buttons ">>" & "<<" the user can adjust his/her preferences for the favorite interests. 

Button "Ok" returns the user to the "Welcome" form, having admitted the setting change;  button "Cancel" does the same, but without a setting change.  Button "Favorite Advisors" lets the user select his/her preferences for the set of advisors, whose messages are preferable to the user.
