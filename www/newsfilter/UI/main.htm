The Main form

This is the base form.  It allows the user to hang around the interest tree.   For each selected interest, the list of advisors and the list of articles appear immediately.  A double click on a certain article lets the user read the article itself.

The menu bar on the top of the form contains three items:  View, Options, Help.   The "Options" item holds 4 basic functions:  it allows the user sort advisors and messages.  Options->favorites allows to change settings in the User Profile (favorite interests and favorite advisors).  Options->filter lets set additional limitations to the advisor set and message set.

The tool bar has been set for the user's convenience.  Here are some functions it holds:

  • the left list-box contains 5 last visited interests.
  • button "Up" moves the user one level up the interest tree.
  • button "Root" brings the user to the interest tree root.
  • the right list-box contains the list of the favorite interests specified in the "Settings" form
  • the three buttons in the top-right corner let the user view the interest tree in any desired performance (among the 3 specified).  The three samples of these performances are presented at the following destinations:  www.e-spaces.com/unnf/ui1.jpg, www.e-spaces.com/unnf/ui2.gif, www.e-spaces.com/unnf/ui3.gif

By default, all the messages responding to the chosen interests and advisors will be reported to the user.  If the user wishes to set additional conditions for the message set, he/she has to choose "Filter" item in the Menu to enter advanced preferences.
