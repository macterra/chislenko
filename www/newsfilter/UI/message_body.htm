Message Body form

The primal purpose of this form is to show the message selected by the user in the main form.  Besides, it offers some other actions to be done to the current message:

  1. Button "Reply" lets the user reply to the message and send an e-mail to the message author.
  2. Button "Previous" displays the previous message.  This button is locked if the current message is the #1 in the selection.
  3. Button "Next" displays the next message.   This button is locked if the current message is the last one in the selection.
  4. Button "Print" lets the user print the message.
  5. Button "Save" saves the message on the user's computer.
  6. Button "Rate" brings up the "Rate Dialogue Box", which allows the user to assign a certain interest to the current message and to grade the message.
  7. Button "Location" shows the News Server and the News Group, where the message came from.

  9. Button "Help" brings up the program manipulating instructions.
