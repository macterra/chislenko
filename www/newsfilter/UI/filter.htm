Advanced Filtering Settings Dialogue

This form lets the user enter advanced settings. 

By default, all the messages under the selected interests & recommended by the favorite advisors will be shown to the user.  This means that by default only the "filter advisors->favorite advisors" only  check-box is selected.

However, there is a possibility for the user to change the selection of messages.

1.  The user can request the messages recommended by the advisors selected in the Settings form only with a specific rating-- from 1 to 10.(filter advisors-> by rating)  If this check box is selected together with "filter advisors->favorite advisors only", only the messages recommended by the favorite advisors rated within the set limitations will be shown.  Otherwise, the information about the favorite advisors in the settings form will be skipped.

2.  The user can request the messages rated by different advisors with a specific message grade--from 1 to 10 (filter messages->by rating)

3.  The user can request the messages written within a specific time period (filter messages->by date)

Just like in #1, this selection(##2,3) can be inquired from the favorite advisors list from the Settings form or not by selecting or deselecting "filter messages->favorite advisors only"
