Message Rate Dialogue

This Dialogue box allows the user to rate the current open message. 

First of all, the user has to decide, which interest is this message related to.   In the "Inerest tree" list-box there are paths to 5 last visited interests in the interest tree.  This is made for the user's convenience.  If the needed interest hasn't been visited by the user recently, he/she has to find the path in the interest tree in the bottom-left part of the form.  The Rating Edit box is locked until the interest is specified.

After the interest and its path are specified, the user has to grade the message from 1 to 10 and enter his/her grade in the Rating Edit box. 

Button "Ok" returns the user to the Message_Body form upon adding the message rate to the database.  Button "Cancel"  returns the user to the Message_Body form without adding any info to the database.
