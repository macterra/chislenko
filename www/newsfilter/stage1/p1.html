User registration/authoring facility (URAF)

URAF accepts new or updated user registration records and authorizes existing users.
 

URAF presents a user a welcome screen with a request for user name and password. In case user doesn’t have a user name he/she can follow a link to a registration page.

 

On this page user will be asked for a desired user name and password (plus any additional questions we would like) and assigned it if possible.

After authorizing user will be taken to a message selection facility.
